## 🪷 Báo Cáo: skills.sh — Hệ Sinh Thái Agent Skills

### 1️⃣ skills.sh là gì?

**skills.sh** là **npm cho AI coding agents** — một registry mở do **Vercel Labs** tạo ra (ra mắt 20/01/2026). Nơi tập trung các skill packages mà agent có thể cài đặt bằng một dòng lệnh.

**Cách dùng:**
```bash
npx skills add <owner/repo> --skill <skill-name> -a openclaw -g -y
```

### 2️⃣ OpenClaw được support chính thức ✅

Trong danh sách ~50 agents được hỗ trợ, **OpenClaw** có mặt với:
- Flag: `-a openclaw`
- Đường dẫn global: `~/.openclaw/skills/`
- Đường dẫn project: `skills/`

### 4️⃣ Phát hiện QUAN TRỌNG từ Vercel (Jan 2026)

Vercel đã publish một blog so sánh **Skills vs AGENTS.md**:

| Cấu hình | Pass Rate | So với baseline |
|----------|-----------|----------------|
| Baseline (không docs) | 53% | — |
| Skill (mặc định) | 53% | +0pp |
| Skill + explicit instructions | 79% | +26pp |
| **AGENTS.md docs index** | **100%** | **+47pp** |

**Vấn đề của Skills:**
- 56% cases agent **không chịu invoke skill** dù có sẵn
- Agent phải quyết định "có nên dùng skill này không" → thường sai
- Instruction wording ảnh hưởng lớn đến kết quả

**Tại sao AGENTS.md thắng:**
1. Không có bước quyết định — content đã có sẵn trong prompt
2. Luôn available — không cần load async
3. Không có ordering issues

**Kết luận của Vercel:** Skills tốt cho **vertical, action-specific workflows** (mà người dùng chủ động yêu cầu). AGENTS.md tốt cho **general framework knowledge**. Cả hai bổ sung cho nhau.

**B. PUBLISH agent-army lên skills.sh:**

- Clone về repo GitHub riêng, thêm SKILL.md

**C. TIẾP TỤC dùng AGENTS.md + MEMORY.md:**

- Skills là **phần bổ sung**, không phải thay thế

**D. THÊM workspace skills:**
- Có thể tạo thư mục `skills/` trong workspace để override bundled skills
- Định dạng: `skills/<tên-skill>/SKILL.md` với YAML frontmatter
